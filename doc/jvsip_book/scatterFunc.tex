\pyComment{
\item{Parameter \ttbf{in} is a view of shape vector and precision \emph{\ttbf{p}}}
\item{Parameter \ttbf{out} is a view with precision \emph{\ttbf{p}}}
\item{Parameter \ttbf{indx} is a view of shape vector and type \ttbf{vi} or \ttbf{mi} depending on the shape of view \ttbf{out}.}
\item{The length of \ttbf{in} and \ttbf{indx} must be the same length.}
\item{Elementwise an index is selected from \ttbf{indx} and a value is selected from \ttbf{in}. The value is placed into \ttbf{out} at the indexed location.}
\item{The user of the function is responsible to ensure indices in \ttbf{indx} are valid for \ttbf{out}}
